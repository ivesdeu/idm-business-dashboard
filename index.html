Developer / setup
Embedded from my-agent-next (agent my-agent). The iframe loads /chat?embed=1 for a headerless panel. Local: cd my-agent-next && npm install && npm run dev, API_KEY_21ST in .env.local, then Refresh panel. Production: set CHAT_EMBED_URL on Netlify (build) to your HTTPS /chat URL so chat-embed-url is not localhost.
